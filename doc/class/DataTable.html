Overview

. (void) expand(vars = {}, unique = true, nreps = 1)

create a table consisting of all combinations of the values in a set of variables

Parameters:

  • vars (Object) — a set of variables indexed by key
  • unique (Boolean) — ensure that all elements in any of the variables are distinct before expanding
  • nreps (Integer) — if greater than 1 the table will be replicated nreps times
